## 1. 数据集
> TuSimple Competitions for CVPR 2017 : [https://github.com/TuSimple/tusimple-benchmark](https://github.com/TuSimple/tusimple-benchmark)

## 2. 目标
使用数字图像处理的基本方法实现车道线检测。除图像的IO操作采用OpenCV实现外，数据结构和处理算法使用C++自行实现。
